Sistema multicuenta (v0.4.0)

Gestiona múltiples cuentas de X de forma segura con cambio rápido entre ellas.

Características

  • Detección automática de login en X.com
  • Almacenamiento seguro en macOS Keychain con encriptación AES-256-GCM
  • Cambio rápido entre cuentas desde el menú nativo
  • Identificadores únicos (UUID) por cuenta
  • Migración automática desde versiones anteriores

Uso

  • Agregar cuenta: Menú Cuentas → Agregar Cuenta... o ⌘⇧N
  • Cambiar cuenta: Click en el nombre de usuario en el menú Cuentas
  • Eliminar cuenta: Menú Cuentas → Eliminar Cuenta Activa o ⌘⌫

Sistema de actualizaciones

La aplicación verifica automáticamente nuevas versiones desde GitHub al iniciar.

Proceso automático

  • Verificación al iniciar (2 segundos después del arranque)
  • Consulta a GitHub API: github.com/686f6c61/Xcom-mac-silicon/releases
  • Comparación de versiones (actual vs. última disponible)
  • Apertura automática de la página de releases si hay actualización

Verificación manual

Accede a Menú Ayuda → Buscar Actualizaciones para verificar manualmente.

Seguridad y privacidad

  • Credenciales almacenadas en macOS Keychain (protección del sistema operativo)
  • Encriptación AES-256-GCM para todos los datos sensibles
  • Derivación de claves con Argon2id (resistente a ataques)
  • Identificadores únicos UUID v4 por cuenta
  • Sin recopilación de datos personales
  • Sin intermediarios: comunicación directa con X.com
  • Código abierto: auditable en GitHub

Arquitectura técnica

Backend

  • Rust (edición 2021)
  • Tauri 2.x (framework multiplataforma)
  • Security Framework (integración con macOS Keychain)
  • AES-GCM 0.10 (encriptación autenticada)
  • Argon2 0.5 (derivación de claves)

Frontend

  • WebView nativo de macOS (WKWebView)
  • JavaScript modular (ES6+)
  • Detector de login automático con monitoreo de DOM
  • Comunicación bidireccional con backend vía postMessage

Menús nativos

  • 7 menús principales: App, Archivo, Edición, Cuentas, Visualización, Navegación, Ayuda
  • 50+ elementos de menú con atajos de teclado
  • Menú Cuentas dinámico (se reconstruye al agregar/eliminar cuentas)
  • Integración completa con macOS (servicios, ocultación, etc.)

Comandos Tauri disponibles

Interfaz JavaScript-Rust para funcionalidades avanzadas:

Gestión de credenciales

  • save_credentials - Guardar credenciales encriptadas
  • get_credentials - Recuperar credenciales desencriptadas
  • delete_credentials - Eliminar credenciales del Keychain

Multicuenta

  • list_accounts - Listar todas las cuentas guardadas
  • get_active_account - Obtener cuenta activa actual
  • set_active_account - Establecer cuenta activa
  • save_account_credentials - Guardar cuenta con detección automática
  • delete_account - Eliminar cuenta permanentemente
  • switch_account - Cambiar a otra cuenta
  • rebuild_accounts_menu - Reconstruir menú de cuentas

Utilidades

  • check_updates - Verificar actualizaciones desde GitHub
  • open_help - Abrir página de ayuda
